R语言—如何下载和使用包

2021-06-02

R语言中有非常多的包，每个包都具有不同的功能。因此，在正式的R语言学习之前，需要学会如何下载和安装R的包，以此进行下一步学习。

R语言中有非常多的包，每个包都具有不同的功能。因此，在正式的R语言学习之前，需要学会如何下载和安装R的包，以此进行下一步学习。

  • install.packages( ) 安装

1.打开RStudio软件，在RStudio中输入代码install.packages()，这里以常用的一个集合包作为示例，因此输入install.packages("tidyverse")，记得 加英文双引号，然后点击Run

2.此时在控制台中就会出现安装过程，直至安装成功

  • Install Packages 安装

1.打开菜单栏，Tools-Install Packages

2.这里有两种下载， 一种是从CRAN，也就是从R的官网直接联网下载， 一种是首先下载R包，直接加载即可

3.直接使用第一种，输入包的名称，点击install即可安装成功

4.或者直接点击右下角的install

5.输入名称，安装即可

  • R包的加载

在安装好R包之后，我们需要加载R包，然后进行R包的使用。在RStudio中输入library( )，即可加载包

在这里，我们加载tidyverse包即可，输入：

library(tidyverse)

Attaching packages里面显示了该集成包中所包含的子包：ggplot2，tibble，tidyr等。其他包的安装也如上。

安装好包之后，就等着教程进行学习吧！
